웹 개발시, localhost를 휴대폰에서도 가능하게 하는법

웹개발 pc가 랜선(이더넷)을 사용할 때:
cmd에서 'ipconfig' 명령어를 입력하고 사용하는 이더넷의 ipv4의 주소를 확인한다.
브라우저에서 '확인한 ipv4:포트번호'를 입력하면 사용할 수 있다.
폰에 있는 브라우저에서도 '확인한 ipv4:포트번호'를 입력하면 사용할 수 있다.

웹개발 pc가 와이파이를 사용할 때:
cmd에서 'ipconfig' 명령어를 입력하고 사용하는 와이파이의 ipv4의 주소를 확인한다.
브라우저에서 '확인한 ipv4:포트번호'를 입력하면 사용할 수 있다.
폰이 pc와 같은 와이파이를 사용하고 있는 상태에서
폰에 있는 브라우저에서도 '확인한 ipv4:포트번호'를 입력하면 사용할 수 있다.

*pc로 웹개발을 할 때 앵귤러, 리액트 등을 사용할 때 서버를 열었음에도 pc또는 폰에서 
'확인한 ipv4:포트번호'를 브라우저에 입력했음에도 들어갈 수 없을 때는 
'ng serve --host 0.0.0.0 또는 ng serve --host 특정ipv4' => 앵귤러
'npm start --host 0.0.0.0 또는 npm start --host 특정ipv4' => 리액트

같이 명령어를 입력하여 서버를 열면 될 수 있다.
